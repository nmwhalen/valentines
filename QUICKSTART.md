## Step 1: Add Your Photos (5 minutes)
1. Prepare 6 photos of you and Leah
2. Rename them to: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, `photo5.jpg`, `photo6.jpg`
3. Copy them into the `images/` folder

## Step 2: Customize the Text (10 minutes)
1. Open `index.html` in a text editor (Notepad, VS Code, etc.)
2. Search for `<p class="caption">` (there are 6 of them)
3. Replace each caption with your own memory description
4. Scroll to the bottom and find `<p class="closing-message">`
5. Write your personal Valentine's message to Leah
6. Edit `<p class="closing-signature">` to sign it however you'd like

## Step 3: Test Locally (2 minutes)
1. Double-click `index.html` to open it in your browser
2. Click "Begin" to see the gallery
3. Click each colored shape to reveal the photos
4. Make sure all photos load correctly
5. Read through to verify all text is correct

## Step 4: Deploy to GitHub Pages (10 minutes)

### Create Repository
1. Go to https://github.com/new
2. Repository name: `valentines-for-leah` (or any name you like)
3. Choose "Public" (or "Private" if you have GitHub Pro)
4. Click "Create repository"

### Upload Files
1. On the repository page, click "uploading an existing file"
2. Drag and drop ALL files from the Valentines folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - The entire `images` folder with your photos
3. Scroll down and click "Commit changes"

### Enable GitHub Pages
1. Click "Settings" at the top
2. Click "Pages" in the left menu
3. Under "Branch", select "main" and click "Save"
4. Wait 1-2 minutes
5. Your site will be live at: `https://YOUR-USERNAME.github.io/valentines-for-leah/`
